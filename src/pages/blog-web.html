What is HTML

Hyper-Text Markup Language (HTML) is a markup language used to create Web content. It was created by Tim Bernes-Lee, the famous computer scientist who invented the World Wide Web. Many software developers begin their career by learning HTML and it forms a strong foundation for web development.


HTML Tags

The HTML tag is a code HTML uses to interpret user instructions. A tag becomes a starting point for coder statements and is enclosed in an angle bracket with the command line in the middle.


    The following list is some examples of HTML tag:

  • <h1> - Header 1

  • <p> - Paragraph

  • <img> - Image

  • <em> - Emphasis

  • <li> - List


HTML Element

HTML element is a structure of code that HTML reads to determine what will be the output on your web browser. It consists of opening tag <h1>, your output, and closing tag </h1>. Once HTML renders an element it will display the equivalent tag/instruction including your output to the web browser.


HTML Attribute

HTML attributes can be identified as additional characteristics of your HTML element. Attributes add additional content to your element and should be included on your opening tag. For example, <img src=”sample.jpg”> this will give the source src=”” for an image tag. Once this element has been rendered, it will display the sample.jpg as the image on your web browser.


Inline Element

Inline is a display property in HTML that manipulates the position of an element. If an HTML element is in line with another element, it will place itself horizontally with each other.


Block Element

Think of a large box that has a single pen inside of it. Even though your content is only a pen, the box will consume all of the spaces inside your house. This goes as well with Block Element, the block will be kind of a box in your website and consume all of the available widths.


CSS Rule

The rule is a command in CSS that modifies the style of an HTML element. Rules can be described as a statement that triggers the display of an element in HTML. CSS mainly focuses on beautifying your plain HTML pages. CSS rules can be added inside the opening tag or, a separate CSS file with selectors.


CSS Value

CSS values are the most important property of CSS. Adding CSS rule without a value to it will not possibly work. When adding a CSS rule we define the output of the HTML element by adding CSS values to it. For example, we added a rule of color to our <h1> earlier and a blue value. CSS values answer the question ‘what’ on our CSS rule.
