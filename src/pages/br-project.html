Project Details

Description

Own your land is a mobile application that offers a route recording feature, route sharing, photo sharing, and badges for completing specified trails. There are several Gamification ideas that give users some achievements they can share with their friends. In collaboration with the product owner and Land Rover MENA, I built and conceptualized the user interface for this application.


I began by interviewing stakeholders and creating significant personas. As a result of gathering all the requirements and conducting UX research, I was able to design the user flow and create the mockups. The badge feature was part of my gamification strategy.
